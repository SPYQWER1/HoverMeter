=== GetPersonalPlan API ===
The spike script (.omo/spike/get_personal_plan.py) is fully implemented with correct
V4 HMAC-SHA256 signature but could not be executed because VOLCENGINE_ACCESS_KEY and
VOLCENGINE_SECRET_KEY environment variables were not set.

The script implements:
- Canonical request with signed headers: host;x-content-sha256;x-date
- Credential scope: {YYYYMMDD}/cn-beijing/ark/request
- Signing key derivation: kSecret -> kDate -> kRegion -> kService -> kSigning
- Proper X-Date format (YYYYMMDDTHHMMSSZ) and X-Content-Sha256 (SHA256 hex of body)
- Response capture to .omo/evidence/task-0-spike-response.json
