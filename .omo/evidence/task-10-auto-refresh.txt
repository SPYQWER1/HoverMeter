Hook behavior:
- Initial fetch on mount (useEffect with fetchData(true))
- setInterval with configurable interval (default 5 min = 300000ms, floor 30000ms)
- Interval cleared on unmount via cleanup function
- fetchData(true) sets loading=true, fetchData(false) sets refreshing=true
- Volcano usage always fetched (arkcli handles auth independently)
- DeepSeek balance only fetched when deepseek_key credential exists
- On refresh, credentials are re-loaded to pick up changes from settings

Manual refresh:
- Refresh button (↻) in title bar next to settings gear
- Calls refresh() which triggers fetchData(false)
- Button spins (CSS animation) while refreshing, disabled to prevent double-clicks
