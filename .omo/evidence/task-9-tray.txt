Date: 2025-06-23

## Implementation (src-tauri/src/lib.rs)

### Tray Menu Items
- Show Widget (id: "show") -> shows main window + sets focus
- Settings (id: "settings") -> shows main window + emits "show-settings" event + sets focus
- Separator (PredefinedMenuItem::separator)
- Quit (id: "quit") -> app.exit(0)

### Tray Left-Click Behavior
- show_menu_on_left_click(false) set so left-click does NOT open context menu
- on_tray_icon_event handler matches TrayIconEvent::Click with MouseButton::Left + MouseButtonState::Up
- Toggles main window: if visible -> hide(), if hidden -> show() + set_focus()

### Close-to-Tray
- on_window_event handler matches WindowEvent::CloseRequested
- Calls api.prevent_close() to prevent actual window close
- Calls window.hide() to hide window to tray
- App continues running in system tray

### Tray Icon
- ID: "main-tray"
- Icon: app.default_window_icon() (from tauri.conf.json bundle icons)
- Tooltip: "HoverMeter"
