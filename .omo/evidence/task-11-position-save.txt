Task 11: Window Position Save/Restore Verification
====================================================
Date: 2025-06-23

1. tauri-plugin-window-state registration (Rust)
-------------------------------------------------
src-tauri/src/lib.rs line 27:
  .plugin(tauri_plugin_window_state::Builder::default().build())

The plugin is registered with default settings. It auto-saves window
state (position, size, maximized, fullscreen) on window events:
  - WindowEvent::Moved
  - WindowEvent::Resized
  - WindowEvent::CloseRequested / Destroyed

The plugin also restores saved state on app startup, overriding the
"center": true default in tauri.conf.json when a saved position exists.

2. Explicit saveWindowState on move/resize (frontend)
------------------------------------------------------
src/App.tsx mount useEffect:

  const unlistenMoved = win.onMoved(() => {
    saveWindowState(StateFlags.ALL).catch(...);
  });
  const unlistenResized = win.onResized(() => {
    saveWindowState(StateFlags.ALL).catch(...);
  });

Imports:
  import { saveWindowState, StateFlags } from "@tauri-apps/plugin-window-state";

StateFlags.ALL = 63 (SIZE|POSITION|MAXIMIZED|VISIBLE|DECORATIONS|FULLSCREEN)

This provides belt-and-suspenders coverage on top of the plugin's
auto-save. The window is resizable:false, so onResized won't fire
in normal use, but onMoved fires on every window drag.

3. Capabilities
----------------
src-tauri/capabilities/default.json includes:
  "window-state:default"

This permission set allows the frontend to call saveWindowState.

4. Close-to-tray interaction
-----------------------------
The window is hidden (not closed) on X button click (Task 9:
on_window_event handles CloseRequested -> prevent_close + hide).
The window-state plugin's auto-save on close won't trigger for
hide-to-tray, but the onMoved handler ensures position is saved
during dragging, and the plugin saves on app exit (tray Quit).

5. Cleanup
-----------
All three listeners (show-settings, onMoved, onResized) are cleaned up
in the useEffect return:
  return () => {
    unlistenSettings.then((fn) => fn());
    unlistenMoved.then((fn) => fn());
    unlistenResized.then((fn) => fn());
  };
